Bản HTML giới thiệu của PAT-Edu

Từ file tĩnh này, bạn có thể mở đúng hệ thống PAT-Edu đang chạy bằng PHP và MySQL.

File index.html dùng để mô tả nhanh cấu trúc triển khai. Hệ thống chính chạy ở index.php với giao diện trong assets/css/pat_edu.css, hành vi trong assets/js/pat_edu.js và cơ sở dữ liệu tại database/pat_edu_v2.sql.

HTML

Trang giới thiệu

Dùng khi mở trực tiếp file để người xem biết phải vào localhost để thấy hệ thống chạy thật.

CSS

Giao diện pastel

Toàn bộ giao diện mới dùng tông xanh pastel, trắng và xanh than sạch, dịu và chuyên nghiệp.

JS

Tự làm mới thông báo

JavaScript chịu trách nhiệm lọc dữ liệu trên giao diện và tự tải thông báo mới cho phụ huynh, học sinh.

DB

Cơ sở dữ liệu mở rộng

Schema mới có đầy đủ người dùng, khóa học, thông báo, bài nộp, Zalo và phản hồi chấm bài.
